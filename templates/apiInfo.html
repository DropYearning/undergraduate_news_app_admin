  用户相关API接口

（1）登陆

方法

GET

URL

/user/{username}/{password}

返回码

500：登陆成功

501：用户名尚未注册

502：用户名和密码不正确

503：发生未知错误

（2）注册

方法

POST

URL

/user/{username}/{password}

返回码

600：注册成功

601：用户名已被占用

（3）添加收藏

方法

POST

URL

/savenews/{username}/{news_id}

返回码

700：收藏成功

701：该新闻已收藏

（4）取消收藏

方法

DELETE

URL

/savenews/{username}/{news_id}

返回码

800：已取消

801：该用户未收藏该新闻

（5）添加访问历史记录

方法

POST

URL

/userhis/{username}/{news_id}

返回码

不设置返回码

  新闻列表相关API接口

（1）获取某频道某页的所有新闻数据（默认一页10篇新闻）

方法

GET

URL

/newslist/{channelname}/page=?{idx }

返回值

带有该页所有新闻的JSON格式数据

（2）获取某条新闻的内容详情

方法

GET

URL

/detail/{channelname}/{news_id}

返回值

带有该新闻详细内容的JSON格式数据

（3）获取某用户收藏的所有新闻列表

方法

GET

URL

/savedlist/{username}

返回值

带有该用户收藏的所有新闻的JSON格式数据

（1）获取某用户访问过的所有新闻列表

方法

GET

URL

/viewlist/{username}

返回值

带有该用户访问过的所有新闻的JSON格式数据

  新闻推荐相关API接口

（1）为未登录用户随机推荐新闻

方法

GET

URL

/rcmrandom/

返回值

带有推荐结果新闻列表的JSON格式数据

（2）为已登录用户个性化推荐新闻

方法

GET

URL

/rcm/{username}

返回值

带有推荐结果新闻列表的JSON格式数据

（3）根据一条新闻的ID推荐与之相似的新闻

方法

GET

URL

/rcdid/{channelname}/{news_id}

返回值

带有推荐结果新闻列表的JSON格式数据
